### Opciones de Juego
- 1) Dos jugadores en el teclado

    - Teclas
      - w - Arriba
      - s - Abajo
  - Jugador 2
    - Teclas
      - Flecha de Direccion Arriba
      - Flecha de Direccion Abajo
- 2) Jugar con la maquina
  - Dificultades
    - F) Facil
    - D) Dificil
    - E) Extremo(Solo para gente del tics)
